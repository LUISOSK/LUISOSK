Ecuacion cuadratica

Una ecuación de segundo grado o ecuación cuadrática de una variable es aquella que tiene la expresión general:

Ecuacion de segundo grado

a * x ^ 2 + bx + c = 0 a ≠ 0

Donde x es la variable; mientras que a, b y c son valores constantes y corresponde con los coeficientes:

  1. a. cuadrático (a)
  2. b. lineal (b)
  3. c. independiente (c)

Para una ecuación cuadrática con coeficientes reales o complejos existen siempre dos soluciones (x, y x2), no necesariamente distintas, llamadas raíces, que pueden ser reales o complejas. Fórmula general para la obtención de raíces: x = (- b plus/minus sqrt(b ^ 2 - 4ac)) / (2a) Se utiliza el símbolo + para indicar las dos soluciones.

C++

El primer programa que se suele crear al comenzar con un lenguaje de programación consiste en escribir algo en pantalla. Es muy frecuente que ese texto que escribe el programa sea un saludo al mundo que le rodea, un "Hola, mundo". En C++ seria así;

#include iostream

std::cout << "Hola, mundo";

return 0;

}

Si no has programado antes, este programa debe sonar casi tan ilegible como si estuviese escrito en chino. Aun así, vamos a empezar a acercarnos al significado de cada frase sin entrar en mucho detalle:

  • #include deberá aparecer al principio de cualquier programa que escriba en pantalla o lea desde teclado
  • int main () indica que lo que viene a continuación es la parte principal, el cuerpo de nuestro programa
  • {y} son las llaves, que indican el principio y el final de un bloque, en este caso del cuerpo de nuestro programa
  • std::cout << "Hola, mundo"; es la única orden real que tiene nuestro programa por ahora. Es la orden que se encarga de escribir en pantalla lo que indiquemos entre comillas
  • rteturn 0; sirve para indicar que el programa ha terminado sin errores
